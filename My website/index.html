Project description

This project is a Java-based desktop application designed to manage an E-Book Store. It provides a graphical user interface (GUI) for managing books, publishers, and authors in a database. The application uses AWT (Abstract Window Toolkit) for the GUI and JDBC (Java Database Connectivity) for database operations.
